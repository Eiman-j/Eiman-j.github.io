The Requisition Ecosystem

The Smart Classroom AI Portal is a comprehensive requisition platform where students and faculty can reserve classrooms and labs seamlessly.

  • Relational Database: Linked via SQLAlchemy to manage Rooms, Schedules, and Bookings.
  • Automated Governance: The "6 PM Rule" automatically flags evening slots for manual Admin approval.
  • Multi-Portal Access: Dedicated dashboards for Admins, Faculty, and Students.

Team Members: Eiman, Aqsa Zeeshan, Arfah Habib, Arsha Imran

Database Schema

The system maintains data integrity across four core tables: Users, RoomsList, SemesterSchedule, and BookingsNew. This relational structure allows the AI to cross-reference weekly class patterns with individual booking requests in real-time.

AI & System Functionalities

CampusBuddy AI

A Gemini-powered chatbot that summarizes feedback and checks live room availability using database context.

ML Demand Forecast

Uses Linear Regression to analyze booking history and project future demand trends for the coming week.

AI Maintenance

LLM-based sentiment analysis flags critical equipment issues (AC, Projectors) from student messages.

Peak Hour Detection

ML-driven traffic analysis identifies congested hours to optimize user booking times.
