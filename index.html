Smart Classroom AI

An Intelligent Facility Management System driven by LLMs and Predictive Analytics.

Project Overview

This project transforms traditional classroom management into a smart ecosystem. By combining real-time database tracking with Machine Learning, we provide students and admins with predictive insights, automated scheduling, and an AI-powered campus assistant.

Core AI Functionalities

CampusBuddy

A "chill" LLM chatbot using Gemini 2.5 Flash Lite to assist students with real-time room availability and feedback summaries.

Demand Forecast

Linear Regression models analyze historical booking data to predict sharp rises in demand for the upcoming week.

Peak Hour ML

Identifies peak-traffic hours by fitting daily usage patterns into a predictive model, helping users avoid crowded slots.

AI Maintenance

Uses Gemini to perform sentiment analysis on messages, automatically flagging critical equipment issues for admins.
